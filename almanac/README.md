# Almanac — vertical video engine inside templatize-jonops

This directory contains the **Almanac** vertical-video content pipeline, integrated into the per-brand JonOps container template.

Almanac replaces the older Remotion-based path (`generate-tiktok-video.sh` + `remotion/render-ffmpeg.py`) with a HyperFrames + GSAP composition engine that has karaoke per-word captions, beat-synced cuts, the warm-hug content doctrine, and `verified_facts` injection. ~$0.33 per 30-second 9:16 vertical video.

Standalone repo: https://github.com/jonjonesai/almanac

## Container layout

Each JonOps container is one brand. The Almanac engine + that brand's config live under `/opt/jonops/almanac/`:

```
/opt/jonops/almanac/
├── almanac_pipeline.py      # orchestrator (shared engine, brand-agnostic)
├── docs/                    # philosophy, verified_facts pattern, brand onboarding
├── templates/               # brand.json, DESIGN.md, dispatcher.yml templates
├── LICENSE                  # MIT
└── <brand-slug>/            # ← provisioned per container by /init wizard
    ├── brand.json           # voice, colors, fonts, Airtable IDs, anti-patterns
    ├── DESIGN.md            # visual identity manifesto
    ├── hyperframes.json     # HyperFrames project config
    ├── meta.json            # HyperFrames metadata
    ├── package.json         # node deps for hyperframes render
    ├── scripts/
    │   ├── build-karaoke-html.py     # per-brand HTML+GSAP composer
    │   └── <brand>_topic_researcher.py  # optional: seeds Airtable from real sources
    ├── assets/              # populated per fire (gitignored)
    └── renders/             # populated per fire (gitignored)
```

## How it's invoked

The `social-tiktok.md` skill, when called in VIDEO mode by `social-poster-2.md`, routes through Almanac:

```bash
python3 /opt/jonops/almanac/almanac_pipeline.py \
  --brand <brand-slug>
```

The pipeline reads the next `Status=Queued` row from the brand's Airtable Social Queue, generates a 7-beat warm-hug script (with the verified_facts contract), produces media via Freepik/FLUX + ElevenLabs voice + Replicate WhisperX timing + Replicate MusicGen background score, renders via HyperFrames, uploads to Cloudinary, and returns the video URL for downstream Metricool scheduling.

This replaces the old `generate-tiktok-video.sh` (Remotion slideshow path). One render, two platforms — TikTok and Instagram Reels share the MP4. Facebook + Pinterest static images are still generated by the existing `social-facebook.md` + `social-pinterest.md` subskills.

## Why the rename

Was `sp2-hyperframes`. Renamed to **Almanac** 2026-05-16 to reflect what the system actually does: each brand has its own *almanac* — a structured spine of dates, ephemerides, anniversaries, cultural calendars — and the pipeline draws daily readings from it.
