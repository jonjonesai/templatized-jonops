# Brand Assets

Store your business's brand assets here. These are not used by JonOps skills directly — they're just a home for your brand files.

## Directory Structure

```
assets/
├── logos/      # Brand logos (PNG, SVG)
├── images/     # Product images, hero images, etc.
├── fonts/      # Custom brand fonts (if any)
└── README.md   # This file
```

## Notes

- These files are gitignored by default (except .gitkeep)
- Upload brand assets after running `/init`
- Reference these in CLAUDE.md if skills need to use them
