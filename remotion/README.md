# Remotion FFmpeg Renderer

This directory contains the FFmpeg-based video renderer for the TikTok/Reels pipeline.

## Files
- `render-ffmpeg.py` — Main renderer. Takes a JSON config and produces an MP4 with Ken Burns zoompan, subtitles, title/outro cards, BGM, and voiceover.
- `public/` — Working directory for slide images, BGM, voiceover, and cache.

## FFmpeg Requirement
The renderer needs a full FFmpeg build (not stripped). If the system FFmpeg is missing video filters, install a static build:
```bash
# Download static FFmpeg 7.x
wget -O /home/agent/local-libs/ffmpeg https://johnvansickle.com/ffmpeg/releases/ffmpeg-release-amd64-static.tar.xz
# render-ffmpeg.py auto-detects /home/agent/local-libs/ffmpeg if present
```

## Usage
```bash
python3 render-ffmpeg.py --config /tmp/config.json --public-dir ./public --out-dir ./out
```

The config JSON is generated by `generate-tiktok-video.sh` — you typically don't call this directly.
